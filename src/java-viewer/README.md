This example demonstrates use of Magnum through JNI interface from Java
application. It is based on blog post from http://www.twodee.org/blog/?p=189
and CMake Java integration from http://blog.cryptomilk.org/2011/01/15/cmake-java-support/ .

Usage
-----

After compilation run the JAR file from terminal:

    java -Djava.library.path=. -jar JNativeCanvas.jar
